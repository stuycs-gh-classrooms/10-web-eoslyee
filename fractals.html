Koch Curve

Basic Curve

This is the Original Koch Curve.
draw_koch(patrick, 4, 20)

My Koch Curve

draw_koch(patrick, 5, 20, 5)
My modifications are:

  • The color of each koch curve is randomized to make a rainbow.
  • There has been two more lines of koch curve code added, so that when the function is called, koch curve is called six times instead of four.
  • A new parameter that allows you to adjust the width of the pen line has been added.



  def draw_koch(t, depth, length, width):
     if depth == 1:
         t.color(randint(100, 230),
                 randint(100, 230),
                 randint(100, 230))
         t.fd(length)
     elif depth > 1:
         draw_koch(t, depth - 1, length, width)
         t.lt(60)
         draw_koch(t, depth - 1, length, width)
         t.rt(120)
         draw_koch(t, depth - 1, length, width)
         t.lt(60)
         draw_koch(t, depth - 1, length, width)
         t.rt(120)
         draw_koch(t, depth - 1, length, width)
         t.lt(60)
         draw_koch(t, depth - 1, length, width)

Sierpinski Triangle

Basic Sierpinski Triangle

This is the Original Triangle.
draw_sierpinski(patrick, 6, 200)

My Sierpinski Triangle

draw_sierpinski(patrick, 4, 300, 5)
My modifications are:

  • Each triangle has been filled with a randomized color.
  • At the end of every sierpinski triangle with depth > 1, a circle is draw from the starting point.
  • A new parameter that allows you to adjust the width of the pen line has been added.
  • The triangle function itself has not been modified.



  def draw_sierpinski(t, depth, length, width):
    t.fillcolor(randint(175, 230),
                randint(175, 230),
                randint(175, 230))
    if depth == 1:
      t.begin_fill()
      triangle(t, length)
      t.end_fill()
    else:
      draw_sierpinski(t, depth-1, length / 2, width)
      t.fd(length / 2)
      draw_sierpinski(t, depth-1, length / 2, width)
      t.bk(length / 2)
      t.lt(60)
      t.fd(length / 2)
      t.rt(60)
      draw_sierpinski(t, depth-1, length / 2, width)
      t.rt(120)
      t.fd(length / 2)
      t.lt(60)
      t.circle(length/2)
      t.lt(60)

Tree

Basic Tree

This is the Original Tree.
tree(patrick, 4, 50, 20)

My Tree

tree(patrick, 5, 50, 20, 5)
My modifications are:

  • The function blossom(t, size, width) has been added, which will draw a cherry blossom. This is called when dept = 0.
  • Each tree will have a cherry blossom at the very tip of the branch, while the other branches are colored brown.
  • The angle and the length of each branch has been randomized
  • As the tree gets more complex, the branches become thinner and shorter
  • The width parameter has been added



  def blossom(t, size, width):
    t.width(width)
    t.lt(30)
    t.pencolor("#446b33")
    t.fd(size)
    t.rt(60)
    t.pencolor("pink")
    t.fd(size)
    t.rt(120)
    t.fd(size)
    t.rt(60)
    t.pencolor("#446b33")
    t.fd(size)
    t.rt(150)
  


  def tree(t, depth, length, angle, width):
    t.width(width)
    if depth == 0:
        blossom(t, length, width)
        t.pencolor("brown")
    else:
        t.pencolor("brown")
        angle += random.randrange(-5, 5)
        length += random.randrange(-5, 5)
        t.fd(length)
        t.rt(angle)
        tree(t, depth - 1, length * 0.75, angle, width * 0.9)
        t.lt(angle * 2)
        tree(t, depth - 1, length * 0.75, angle, width * 0.9)
        t.rt(angle)
        t.bk(length)
